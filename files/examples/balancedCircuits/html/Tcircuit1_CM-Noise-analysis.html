CM Noise analysis

Matrix equation:

\begin{equation} \left[\begin{matrix}I1N + I1P + I_{noise R1N} - I_{noise R1P}\\- I_{noise R1N} + I_{noise R1P} + I_{noise R2}\end{matrix}\right]=\left[\begin{matrix}\frac{2}{R_{a}} & - \frac{2}{R_{a}}\\- \frac{2}{R_{a}} & \frac{1}{R_{c}} + \frac{2}{R_{a}}\end{matrix}\right]\cdot\left[\begin{matrix}V_{in C}\\V_{C}\end{matrix}\right] \end{equation}

Symbolic noise analysis results

Detector-referred noise spectrum

$$S_{out}=2 R_{a} T k + 4 R_{c} T k + S_{A} \left(0.5 R_{a} + R_{c}\right)^{2} + S_{B} \left(0.5 R_{a} + R_{c}\right)^{2}\, \mathrm{\left[\frac{V^2}{Hz}\right]}$$

Source-referred noise spectrum

$$S_{in}=\frac{2 R_{a} T k}{\left(0.5 R_{a} + R_{c}\right)^{2}} + \frac{4 R_{c} T k}{\left(0.5 R_{a} + R_{c}\right)^{2}} + S_{A} + S_{B}\, \mathrm{\left[\frac{A^2}{Hz}\right]}$$

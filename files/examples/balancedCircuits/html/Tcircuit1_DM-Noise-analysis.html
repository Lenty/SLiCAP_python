DM Noise analysis

Matrix equation:

\begin{equation} \left[\begin{matrix}- 0.5 I1N + 0.5 I1P - 0.5 I_{noise R1N} - 0.5 I_{noise R1P}\end{matrix}\right]=\left[\begin{matrix}\frac{0.5}{R_{a}}\end{matrix}\right]\cdot\left[\begin{matrix}V_{in D}\end{matrix}\right] \end{equation}

Symbolic noise analysis results

Detector-referred noise spectrum

$$S_{out}=R_{a}^{2} S_{A} + R_{a}^{2} S_{B} + 8 R_{a} T k\, \mathrm{\left[\frac{V^2}{Hz}\right]}$$

Source-referred noise spectrum

$$S_{in}=0.25 S_{A} + 0.25 S_{B} + \frac{2 T k}{R_{a}}\, \mathrm{\left[\frac{A^2}{Hz}\right]}$$
